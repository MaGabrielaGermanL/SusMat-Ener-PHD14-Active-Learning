## Project Overview
This repository contains the research software and code developed for **PHD14: Active learning for nanostructured materials for energy harvesting**, as part of the **SusMatEner** Doctoral Network. 

The project aims to implement automated active learning loops to optimize the design and performance of nanostructured materials for energy harvesting applications through iterative AI/ML model refinement.

## Key Objectives
* Develop AI/ML models for predicting properties of nanostructured materials.
* Implement active learning workflows to guide experimental and simulation data collection.
* Optimize energy harvesting efficiency using data-driven approaches.

## Interconnections
This work is closely linked to other research outputs within the SusMatEner consortium, specifically:
* **PHD11**: Photonic nano-structuring for solar energy harvesting.
* **PHD12**: New heterostructures for self-powered UV photodetection.
* **PHD13**: Combinatorial optimization of 3D nanostructured piezoelectric harvesters.
